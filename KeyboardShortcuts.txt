These shortcuts will be recognized from any keyboard:

⇧ (hold)		Draw straight line
⌥⇧⌘T		Create text box
⌥↵			While typing, create new line



These shortcuts must come from a Wacom tablet, such as via an express key:

⌘Z			Undo
⇧⌘Z		Redo
⌘Z (hold) 		Clear drawing
⌘D			Switch color
F14 (hold)		Use pen as mouse
